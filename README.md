# ArenaReset

A lightweight **Paper/Folia** plugin that resets game arenas by replacing a WorldGuard area using a schematic (DountSMP Style)


## Dependencies

| [FastAsyncWorldEdit (FAWE)](https://www.spigotmc.org/resources/fastasyncworldedit.13932/) | ✅ |
| [WorldGuard](https://dev.bukkit.org/projects/worldguard) | ✅ |



## Command

| Command | Description | Permission |
|---------|-------------|------------|
| `/arena reset <region1-10>` | Resets the specified arena | `arenareset.reset` |


## Compatibility

| Platform | Supported |
|----------|-----------|
| Paper 1.21+ | ✅ |
| Folia 1.21+ | ✅ |
